# 项目截图

将前端游戏运行后的截图放入此目录，用于 README 展示。

## 推荐截图

| 文件名 | 建议内容 | 用途 |
|--------|----------|------|
| `game-main.png` | 游戏主界面，展示江南水乡场景和 NPC | 放在 README 效果预览区 |
| `game-dialog.png` | NPC 对话界面，展示打字机效果和输入框 | 放在 README 效果预览区 |
| `architecture.png` | 系统架构图（可选，也可用 Mermaid） | 补充说明系统架构 |

## 使用方式

README 中通过相对路径引用：

```markdown
![游戏主界面](./docs/images/game-main.png)
```

> 注意：图片建议控制在 200KB 以内，格式推荐 PNG 或 WebP。
